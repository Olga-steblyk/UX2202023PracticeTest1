Difference between writing in google docs and in HTML documents

In google docs, it is primarily used for word typing and documents. The information can be saved in multiple different ways, such as PDFs and DOCX files, and which can be read more easily. It is also traditionally made for printing documents, portrait form as well, but it is subject to change. Writing in google docs is more comfortable for users in putting in like tabels and images, as it is more streamlined and just a few click away, compared to a HTML doc.


While HTML documents are made to support coding in HTML language. The HTML translates the tags into proper format and structure to display the information. Usually writing in HTML documents, the code is displayed in different colors which all have different meanings. There are nuances that make coding in HTML harder compared to writing in a google doc, as in having the code all work and and run properly, if a mistake were to be made, like a missing ending tag, all of the code will be turned in to the missing tags format.


With the differences listed above, it displays that their is a large difference between writing in google docs and in html.
